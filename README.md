# Monopoly

Jeu du monopoly en JAVA avec une interface graphique.



## Comment jouer

Le jeu ne dispose pas encore d'executable stable.

```bash
  Pour jouer, lancez le projet sous Eclipse avec JavaFX pré-installé.
```
